# Branding & Design System

## Global Design Rules
- **Primary Color**: `#6D214F` (Used for branding and hover states)
- **Shadows**: `box-shadow: rgba(0, 0, 0, 0.35) 0px 5px 15px;`
- **Border Radius**: `rounded-2xl` (default), `rounded-3xl` (large)

## Interaction Rules
- **Hover Transitions**: `transition-all duration-300`
- **Hover Effects**: 
  - `hover:border-[#6D214F]`
  - `hover:scale-[1.01]`
  - `hover:-translate-y-1`

## Typography
- **Primary Fonts**: Inter, Poppins, or Urbanist
- **Headings**: `font-bold`, `tracking-wide`, `text-slate-100`

## Components
- **Sidebar**: Dark glassmorphism, 280px (expanded) / 90px (collapsed)
- **Buttons**: Smooth gradients, icon support, loading/disabled states.
- **Cards**: Border glow, shadow effect, rounded corners.
