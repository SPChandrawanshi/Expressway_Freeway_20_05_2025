# Frontend Architecture & Technology Stack

## Tech Stack
- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **UI Components**: ShadCN UI
- **Charts**: Recharts
- **Maps**: React Leaflet / Google Maps
- **Tables**: AG Grid
- **Icons**: Lucide React / React Icons

## Folder Structure
```text
src/
 ├── assets/        # Images, logos, global assets
 ├── components/    # Reusable UI components (cards, charts, etc.)
 ├── layouts/       # Main layout structures (DashboardLayout, AuthLayout)
 ├── pages/         # Role-specific dashboard pages
 ├── routes/        # Application routing logic
 ├── services/      # API calls and external services
 ├── hooks/         # Custom React hooks
 ├── context/       # Global state management
 ├── animations/    # Framer Motion variants and presets
 ├── utils/         # Helper functions
 └── App.jsx        # Root component
```
